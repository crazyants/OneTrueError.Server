Create application

You need to start by creating an application. It corresponds to a specific binary in your project.

For a client/server applications you might want to track errors for the server and client separately. To do so you need to create two applications in codeRR. For instance one named "YourProduct.Server" and one named "YourProduct.WpfClient".

You can at any time get support by choosing 'support' in the top right menu.
