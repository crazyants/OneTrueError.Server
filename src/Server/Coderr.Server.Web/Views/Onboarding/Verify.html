Generating an exception

If you want to test the automated handling (unhandled exceptions), the easiest way is just to add a throw somewhere. 

throw new InvalidOperationException("Something for codeRR to bite in.");

Remember to visit the page/function that throws that exception :)

Manually reporting exceptions

If you have not activated the handling of uncaught exceptions you can use a try/catch to verify:

try
{
    throw new InvalidOperationException("Something for codeRR to bite in.");
}
catch (Exception ex)
{
    Err.Report(ex);
}
