Configure your application

The codeRR client is installed through nuget. There is a core package that can be used to manually report exceptions no matter what kind of .NET application you have. The true power comes with the integration libraries, as they collect a rich set of context information which makes makes it a pure joy to figure out why the exception happened.

Install one of the packages and then click on the correct one below to get configuration instructions.

  • WCF
  • WinForms
  • ASP.NET (vanilla)
  • ASP.NET MVC
