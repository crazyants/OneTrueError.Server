Congratulations on installing codeRR!

Quick start

The first thing you need to do is to add one or more applications. It's done through the hamburger menu top/right. You can also rename "DemoApp" through that menu.

Once done, you can invite team members under the application menu to the left.

About codeRR

codeRR is a client/server application where the clients (nuget packages) detect and capture exceptions together with contextual information (such as thread info, app info etc). That information is then uploaded to the codeRR server for analysis.

If you want to learn more, read our documentation. Features and suggestions are welcome in our community
